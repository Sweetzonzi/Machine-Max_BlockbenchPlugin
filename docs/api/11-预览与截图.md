## 1. Preview 类 — 3D 视口

继承自 [Deletable](file:///d:/Files/Project_MinecraftMods/Machine-Max_BlockbenchPlugin/docs/09-工具与辅助.md#7-deletable-类)。管理 Blockbench 中的 3D 预览视口，每个 Preview 对应一个独立的 3D 渲染画布。

### 属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 视口唯一标识符 |
| `canvas` | `HTMLCanvasElement` | 关联的 Canvas DOM 元素 |
| `width` | `number` | 视口宽度（像素） |
| `height` | `number` | 视口高度（像素） |
| `isOrtho` | `boolean` | 是否为正交投影模式 |
| `angle` | `number \| null` | 当前视角预设索引 |
| `camera` | `THREE.PerspectiveCamera \| THREE.OrthographicCamera` | Three.js 相机对象（只读，返回当前活跃相机） |
| `renderer` | `THREE.WebGLRenderer` | Three.js WebGL 渲染器 |
| `background` | `any` | 背景设置（颜色/图片/场景） |
| `raycaster` | `THREE.Raycaster` | Three.js 射线检测器 |
| `controls` | `any` | 轨道控制器实例 |

### 方法

#### `preview.raycast(event, options?): RaycastResult`

对鼠标/触摸事件进行射线检测，返回与场景的交点信息。

```javascript
var result = preview.raycast(mouseEvent);
if (result.type === 'element') {
    console.log('命中元素:', result.element.name);
}
```

#### `preview.render(): void`

手动触发视口渲染。通常在场景变化后调用以更新显示。

```javascript
preview.render();
```

#### `preview.setProjectionMode(orthographic): void`

设置投影模式。接收布尔值，`true` 为正交投影，`false` 为透视投影。

```javascript
preview.setProjectionMode(true);   // 切换到正交投影
preview.setProjectionMode(false);  // 切换到透视投影
```

#### `preview.setFOV(fov): void`

设置透视相机的视野角度（度）。

```javascript
preview.setFOV(60);  // 设置为 60 度视野
```

#### `preview.setLockedAngle(angle): void`

锁定视角到指定角度预设。传入角度预设的值来锁定视角，不传或传 `null` 解锁。

```javascript
preview.setLockedAngle(0);       // 锁定到默认视角
```

#### `preview.loadAnglePreset(preset): void`

加载视角预设，将相机切换到预设角度。

```javascript
preview.loadAnglePreset('front');  // 前视图
preview.loadAnglePreset('iso');    // 等轴测视图
```

#### `preview.newAnglePreset(data): void`

基于当前相机位置创建新的视角预设。

```javascript
preview.newAnglePreset({
    name: '自定义视角',
    icon: 'fa-camera'
});
```

#### `preview.getFacingDirection(): string`

获取相机当前的朝向方向名称。返回 `'north'`、`'south'`、`'east'`、`'west'` 之一。

```javascript
var direction = preview.getFacingDirection();
// 'north' | 'south' | 'east' | 'west'
```

#### `preview.loadBackground(url): void`

加载并设置视口背景图片。

```javascript
preview.loadBackground('path/to/background.png');
```

#### `preview.occupyTransformer(): this`

占用变换控制器，阻止其他操作干扰正在进行的变换。

#### `preview.showContextMenu(event): this`

在指定事件位置显示视口右键菜单。

### 静态属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `Preview.all` | `Preview[]` | 所有已创建的预览视口数组 |
| `Preview.selected` | `Preview` | 当前被选中的预览视口 |

```javascript
// 遍历所有视口
Preview.all.forEach(function(p) {
    p.render();
});

// 获取当前活动视口
var activePreview = Preview.selected;
```

## 2. RaycastResult 类型

射线检测结果对象，描述射线与 3D 场景的交点信息。

### 属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `type` | `'element' \| 'keyframe' \| 'vertex' \| 'cube' \| 'line' \| 'none'` | 命中类型 |
| `element` | `OutlinerElement` | 命中的元素对象（如 Cube、Mesh） |
| `face` | `number` | 命中的面索引 |
| `position` | `THREE.Vector3` | 命中点的世界坐标 |
| `uv` | `THREE.Vector2` | 命中点的 UV 坐标 |

```javascript
function onMouseClick(event) {
    var hit = preview.raycast(event);
    if (hit.type === 'element') {
        // 选中被点击的元素
        hit.element.select();
    } else if (hit.type === 'none') {
        // 点击空白处取消选中
    }
}
```

## 3. Canvas 命名空间 — 3D 画布场景

`Canvas` 是 Blockbench 3D 场景的核心管理命名空间，负责维护 Three.js 场景、材质、网格、骨骼等渲染资源。

### 属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `Canvas.scene` | `THREE.Scene` | Three.js 主场景对象 |
| `Canvas.materials` | `object` | 场景中所有材质集合 |
| `Canvas.meshes` | `object` | 场景中所有网格集合 |
| `Canvas.bones` | `object` | 场景中所有骨骼辅助对象集合 |
| `Canvas.gizmos` | `THREE.Object3D[]` | 场景中所有辅助工具（gizmo）集合 |
| `Canvas.pivot_marker` | `THREE.Object3D` | 旋转轴心标记对象 |
| `Canvas.show_gizmos` | `boolean` | 是否显示辅助工具 |

### 方法

#### `Canvas.updateView(options): void`

更新当前视图。`options` 可指定需要更新的具体方面。

```javascript
Canvas.updateView({ update_selection: true });
```

#### `Canvas.updateAll(): void`

重新生成整个场景。在模型结构发生重大变化时调用，重建所有网格、材质和辅助元素。

```javascript
Canvas.updateAll();
```

#### `Canvas.clear(): void`

清除场景中的所有对象。

```javascript
Canvas.clear();
```

#### `Canvas.raycast(event): RaycastResult`

在主场景上执行射线检测。

```javascript
var hit = Canvas.raycast(mouseEvent);
```

#### `Canvas.getModelSize(): [number, number]`

计算当前模型在场景中的尺寸，返回 `[width, height]` 数组。

```javascript
var size = Canvas.getModelSize();
```

## 4. PreviewScene 类 — 预览场景

预览场景类，为 3D 视口提供不同的背景环境和光照设置。支持多种预设场景（如工作室、户外等）。

### 构造函数

```javascript
new PreviewScene('studio', {
    name: '工作室场景',
    description: '干净的工作室背景',
    light_color: '#ffffff',
    light_side: 1
});
```

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 场景唯一标识 |
| `options.name` | `string` | 场景显示名称 |
| `options.description` | `string` | 场景描述 |
| `options.light_color` | `string` | 光照颜色（十六进制） |
| `options.light_side` | `number` | 光照方向（0=上, 1=右, 2=下, 3=左） |

### 方法

#### `scene.select(): void`

选中并激活该预览场景。

```javascript
myScene.select();
```

#### `scene.unselect(): void`

取消选中该预览场景。

```javascript
myScene.unselect();
```

### 静态属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `PreviewScene.scenes` | `Record<string, PreviewScene>` | 所有注册的预览场景字典 |
| `PreviewScene.active` | `PreviewScene \| null` | 当前激活的预览场景 |

```javascript
// 切换场景
PreviewScene.scenes.outdoor.select();

// 获取当前场景
var current = PreviewScene.active;
```

## 5. PreviewModel 类 — 预览模型

预览模型类，允许在 3D 视口中显示独立的额外模型（如展示台、道具等）。

### 构造函数

```javascript
new PreviewModel('my_model', {
    cubes: [...],
    texture: '...',
    color: '#ff0000'
});
```

### 参数说明

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `id` | `string` | 模型唯一标识 |
| `options.cubes` | `array` | 立方体定义数组 |
| `options.texture` | `string` | 纹理路径或 DataURL |
| `options.color` | `string` | 纯色颜色值（无纹理时使用） |

### 方法

#### `model.enable(): void`

启用并显示该预览模型。

#### `model.disable(): boolean`

禁用并隐藏该预览模型。返回 `false` 表示当前没有活动场景。

```javascript
if (model.disable() !== false) {
    console.log('模型已隐藏');
}
```

#### `model.update(): void`

更新模型状态（如参数变化后刷新显示）。

```javascript
model.update();
```

## 6. Screencam 命名空间 — 截图与录制

`Screencam` 是 Blockbench 的截图和录制功能命名空间，支持截图预览、全屏截图、GIF 创建、延时摄影等。

### 截图

#### `Screencam.screenshotPreview(preview, options, cb): void`

截取指定预览视口的屏幕内容。

```javascript
Screencam.screenshotPreview(Preview.selected, {
    width: 1920,
    height: 1080,
    transparent: false
}, function(dataUrl) {
    console.log('截图完成:', dataUrl);
});
```

#### `Screencam.fullScreen(options, cb): void`

全屏截图，截取整个 Blockbench 窗口。

```javascript
Screencam.fullScreen({ format: 'png' }, function(dataUrl) {
    // 处理截图数据
});
```

#### `Screencam.screenshot2DEditor(options, cb): void`

截取 2D 纹理编辑器的内容。

```javascript
Screencam.screenshot2DEditor({}, function(dataUrl) {
    // 处理截图数据
});
```

#### `Screencam.cleanCanvas(options, cb): void`

截取不带辅助元素（网格、骨骼标记、gizmo 等）的干净场景。

```javascript
Screencam.cleanCanvas({}, function(dataUrl) {
    // 用于输出纯模型展示图
});
```

### 动画录制

#### `Screencam.createGif(options, cb): void`

创建 GIF 动画。包含帧序列捕捉和 GIF 编码。

```javascript
Screencam.createGif({
    fps: 12,
    duration: 2,  // 秒
    repeat: true
}, function(gifDataUrl) {
    // 使用 GIF 数据
});
```

#### `Screencam.recordTimelapse(options): void`

开始录制延时摄影。持续记录视口变化生成延时视频。

```javascript
Screencam.recordTimelapse({
    interval: 1000,  // 毫秒
    onFrame: function(frameData) {
        // 每帧回调
    }
});
```

#### `Screencam.stopTimelapse(): void`

停止正在进行的延时摄影录制。

```javascript
Screencam.stopTimelapse();
```

### 其他

#### `Screencam.NoAAPreview: Preview`

一个特殊的无抗锯齿（No Anti-Aliasing）预览视口，用于需要清晰像素边界的场景。

```javascript
// 使用无 AA 预览渲染
Screencam.NoAAPreview.render();
```
