## 1. Texture 类 — 纹理

Texture 是 Blockbench 中最核心的类之一，集材质（Material）、纹理（Texture）、图像（Image）功能于一身。每个纹理可以关联一个本地文件或作为内部数据保存在内存中。

### 构造函数

```javascript
new Texture(data?, uuid?)
```

| 参数 | 类型 | 说明 |
|------|------|------|
| data | TextureData (可选) | 纹理初始化数据对象 |
| uuid | string (可选) | 自定义 UUID，不传则自动生成 |

**构造示例：**

```javascript
// 创建一个 64x64 空白纹理并添加到项目
new Texture({
    name: 'my_texture',
    width: 64,
    height: 64,
    path: 'textures/block/my_texture.png'
}).add();

// 创建纹理并指定文件夹和命名空间（用于 Java 版格式）
new Texture({
    name: 'stone',
    folder: 'textures/block',
    namespace: 'minecraft',
    id: '0',
    particle: true
}).add();
```

### TextureData 初始化选项

```javascript
interface TextureData {
    path?: string            // 文件路径
    uuid?: string            // UUID
    name?: string            // 纹理名称
    folder?: string          // 文件夹路径（Java 版格式用）
    namespace?: string       // 命名空间（Java 版格式用）
    id?: string              // 纹理 ID/Key
    particle?: boolean       // 是否作为粒子纹理
    visible?: boolean        // 可见性
    render_mode?: string     // 渲染模式
    render_sides?: string    // 渲染面
    wrap_mode?: string       // 包裹模式
    pbr_channel?: string     // PBR 通道
    group?: string           // 所属纹理组 UUID
    frame_time?: number      // 动画帧时间
    frame_order_type?: string // 帧顺序类型
    frame_order?: string     // 自定义帧顺序
    frame_interpolate?: boolean // 帧间插值
    fps?: number             // 动画播放速度（帧/秒）
    internal?: boolean       // 是否内部纹理
    width?: number           // 宽度
    height?: number          // 高度
    saved?: boolean          // 是否已保存
}
```

### 关键属性

| 属性 | 类型 | 说明 |
|------|------|------|
| uuid | UUID | 纹理唯一标识符 |
| name | string | 纹理名称 |
| path | string (可选) | 文件路径 |
| folder | string | 文件夹路径，用于 Java 版 Block/Item 格式 |
| namespace | string | 命名空间 |

| visible | boolean | 可见性（用于分层纹理模式） |

| render_sides | string | 渲染面：`'auto'`、`'front'`、`'double'` |
| wrap_mode | string | 包裹模式：`'limited'`、`'repeat'`、`'clamp'` |
| pbr_channel | string | PBR 通道：`'color'`、`'normal'`、`'height'`、`'mer'` |
| group | string | 所属纹理组 UUID |
| selected | boolean | 是否被选中 |
| multi_selected | boolean | 是否被多选 |
| width | number | 纹理像素宽度 |
| height | number | 纹理像素高度 |

| fps | number | 动画播放速度（帧/秒），默认 7 |
| source | string | 纹理数据源（路径或 base64） |
| saved | boolean | 是否已保存 |
| error | number | 错误码，0 表示无错误 |

| selection | IntMatrix | 绘制模式下的纹理选区 |
| currentFrame | number | 当前动画帧索引 |
| frameCount | number (只读) | 动画帧总数 |
| display_height | number (只读) | 显示高度（考虑帧动画） |
| ratio | number (只读) | 宽高比 |
| offset | ArrayVector2 (只读) | 画布偏移量 `[0, 0]` |
| material | THREE.ShaderMaterial | Three.js 着色器材质（getter/setter） |

### 关键方法

#### 文件加载

**fromFile(file)** — 从文件对象加载纹理数据

```javascript
/**
 * 从文件对象加载纹理
 * @param {Object} file - 文件对象，包含 name、content(可选)、path(可选)
 * @returns {this}
 */
texture.fromFile({ name: 'stone.png', path: 'textures/block/stone.png' });
```

**fromPath(path)** — 从路径加载纹理（仅桌面应用）

```javascript
/**
 * 从文件路径加载纹理
 * @param {string} path - 文件路径
 * @returns {this}
 */
texture.fromPath('textures/block/stone.png');
```

**fromDataURL(dataURL)** — 从 DataURL 加载纹理

```javascript
/**
 * 从 base64 数据 URL 加载纹理
 * @param {string} data_url - base64 编码的数据 URL
 * @returns {this}
 */
texture.fromDataURL('data:image/png;base64,iVBOR...');
```

**fromJavaLink(link, path_array)** — 从 Java 版格式链接加载

```javascript
/**
 * 从 Java 版 Block/Item 格式链接加载纹理
 * @param {string} link - 资源链接
 * @param {string[]} path_array - 路径数组
 * @returns {this}
 */
texture.fromJavaLink('minecraft:block/stone', ['assets', 'minecraft']);
```

**load(cb?)** — 从当前数据源加载纹理

```javascript
/**
 * 从当前 source 加载纹理数据
 * @param {Function} [cb] - 加载完成回调
 * @returns {this}
 */
texture.load(() => {
    console.log('纹理加载完成');
});
```

**reloadTexture()** — 重新加载纹理（仅桌面应用）

```javascript
texture.reloadTexture();
```

#### 编辑与数据

**edit(callback, options?)** — 编辑纹理内容的封装方法

```javascript
/**
 * 编辑纹理的封装方法。在回调中修改 canvas，自动处理撤销和界面更新
 * @param {Function} callback - 编辑回调，接收 canvas 或 jimp 实例
 * @param {TextureEditOptions} [options] - 编辑选项
 */
texture.edit((canvas) => {
    let ctx = canvas.getContext('2d');
    ctx.fillStyle = '#FF0000';
    ctx.fillRect(0, 0, 16, 16);
}, { edit_name: '绘制红色方块' });
```

**TextureEditOptions 选项：**

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| method | `'canvas' \| 'jimp'` | `'canvas'` | 编辑方式 |
| edit_name | string | — | 撤销操作的名称 |
| use_cache | boolean | — | 是否使用缓存的 canvas/jimp 实例 |
| no_undo | boolean | `false` | 是否不创建撤销点 |
| no_update | boolean | `false` | 是否不更新纹理视觉效果 |
| no_undo_init | boolean | — | 是否不初始化撤销 |
| no_undo_finish | boolean | — | 是否不结束撤销 |

## 2. TextureLayer 类 — 纹理图层

纹理图层始终属于一个纹理，每个图层拥有独立的 HTML Canvas 和 2D 渲染上下文。通过图层可以实现非破坏性编辑。

### 构造函数

```javascript
new TextureLayer(data, texture, uuid?)
```

| 参数 | 类型 | 说明 |
|------|------|------|
| data | TextureLayerData | 图层初始化数据 |
| texture | Texture | 所属纹理对象 |
| uuid | string (可选) | 自定义 UUID |

**构造示例：**

```javascript
let layer = new TextureLayer({
    name: '图层1',
    opacity: 1.0,
    visible: true,
    blend_mode: 'default'
}, texture);
```

### TextureLayerData 初始化选项

```javascript
interface TextureLayerData {
    name?: string                    // 图层名称
    in_limbo?: boolean               // 是否处于 limbo 状态
    offset?: ArrayVector2            // 偏移量 [x, y]
    scale?: ArrayVector2             // 缩放 [x, y]
    opacity?: number                 // 不透明度 (0~1)
    visible?: boolean                // 可见性
    blend_mode?: string              // 混合模式
    image_data?: ImageData           // 图像数据
    data_url?: string                // 数据 URL
}
```

### 属性

| 属性 | 类型 | 说明 |
|------|------|------|
| name | string | 图层名称 |
| uuid | UUID | 图层唯一标识符 |
| texture | Texture | 所属纹理 |
| canvas | HTMLCanvasElement | 图层的独立画布 |
| ctx | CanvasRenderingContext2D | 画布的 2D 渲染上下文 |
| in_limbo | boolean | 是否处于 limbo 状态 |
| img | HTMLImageElement | 图片元素 |
| offset | ArrayVector2 | 图层相对于纹理左上角的偏移量 `[x, y]` |
| scale | ArrayVector2 | 图层缩放（由图层变换工具使用，操作后应重置为 `[1, 1]`） |
| opacity | number | 不透明度（0~1） |
| visible | boolean | 可见性 |
| locked | boolean | 是否锁定 |
| blend_mode | string | 混合模式，详见下表 |

### 混合模式

| 模式 | 说明 |
|------|------|
| `'default'` | 默认正常混合 |
| `'set_opacity'` | 设置不透明度 |
| `'color'` | 颜色模式 |
| `'multiply'` | 正片叠底 |
| `'add'` | 线性减淡（添加） |
| `'darken'` | 变暗 |
| `'lighten'` | 变亮 |
| `'screen'` | 滤色 |
| `'overlay'` | 叠加 |
| `'difference'` | 差值 |
| `'alpha_mask'` | Alpha 遮罩 |

IntMatrix 使用 `Int8Array` 存储矩阵中每个像素的整数值（无符号 8 位）。`override` 属性可为整个区域设置覆盖值，用于纹理选区功能。

```javascript
/**
 * 使用选区遮罩指定的画布上下文。
 * 调用后，后续的绘制操作将只影响选区内区域
 * @param {CanvasRenderingContext2D} ctx - 画布上下文
 * @param {ArrayVector2} [offset=[0, 0]] - 画布偏移量（如图层偏移）
 */
// 将选区作为遮罩应用
texture.selection.maskCanvas(ctx, [0, 0]);
// 此后在 ctx 上的绘制将自动被限制在选区内
ctx.fillStyle = 'red';
ctx.fillRect(0, 0, 64, 64);
ctx.restore();
```

### 完整示例

```javascript
// 创建一个纹理选区
let tex = Texture.selected;
let selection = tex.selection;

// 激活自定义选区
selection.activate();

// 选中纹理的左上角 16x16 区域
for (let x = 0; x < 16; x++) {
    for (let y = 0; y < 16; y++) {
        selection.set(x, y, 1);
    }
}

// 获取选区边界
let rect = selection.getBoundingRect();
console.log(`选区边界: (${rect.ax}, ${rect.ay}) - (${rect.bx}, ${rect.by})`);

// 将选区转换为矩形列表
let boxes = selection.toBoxes();

// 使用选区作为遮罩进行绘制
tex.edit((canvas) => {
    let ctx = canvas.getContext('2d');
    selection.maskCanvas(ctx, [0, 0]);
    ctx.fillStyle = '#FF0000';
    ctx.fillRect(0, 0, tex.width, tex.height);
    ctx.restore();
}, { edit_name: '在选区内绘制' });

// 清除选区
selection.clear();
```

## 6. TextureAnimator 命名空间 — 纹理动画

TextureAnimator 处理纹理动画（精灵图/翻书动画）的播放控制。

### 属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `TextureAnimator.start_timecode` | `number` | 动画开始的时间码（秒） |
| `TextureAnimator.frame_total` | `number` | 总帧数 |

### 方法

**TextureAnimator.playAnimationFrame(texture, tick)** — 播放单帧

```javascript
/**
 * 驱动指定纹理的动画帧更新。在动画循环中调用，
 * 根据时间码和纹理的 frameTime 属性自动计算当前帧
 * @param {Texture} texture - 要播放动画的纹理
 * @param {number} tick - 当前时间戳（用于计算时间差）
 */
TextureAnimator.playAnimationFrame(texture, performance.now());
```

**TextureAnimator.update()** — 更新所有动画纹理

```javascript
/**
 * 更新项目中所有动画纹理的当前帧。
 * 遍历 Texture.all 中所有帧数 > 1 的纹理，
 * 对每个纹理调用 playAnimationFrame
 */
TextureAnimator.update();
```

**TextureAnimator.reset()** — 重置所有纹理动画

```javascript
/**
 * 将所有动画纹理的 currentFrame 重置为 0
 */
TextureAnimator.reset();
```

### 纹理动画属性

纹理的动画相关属性通过 Property 系统定义在 `Texture` 类上：

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `texture.currentFrame` | `number` | `0` | 当前播放的帧索引 |
| `texture.frameCount` | `number` (只读) | — | 纹理的总帧数（由纹理高度 / 帧高度计算得出） |
| `texture.frameTime` | `number` | `1` | 每帧的显示时间（秒），通过 Property 系统注册 |
| `texture.fps` | `number` | `7` | 动画播放速度（帧/秒） |

### 完整示例

```javascript
// 创建动画纹理（多帧精灵图）
var tex = new Texture({
    name: 'animated_tex',
    width: 64,
    height: 128,   // 2 帧，每帧 64x64
    frameTime: 0.5  // 每帧 0.5 秒
}).add();

// 检查纹理动画属性
console.log('帧数:', tex.frameCount);
console.log('帧速率:', tex.fps, 'fps');

// 手动控制帧
tex.currentFrame = 1;  // 切换到第二帧
tex.updateMaterial();

// 在渲染循环中驱动动画
function onRenderFrame() {
    TextureAnimator.update();
}

// 重置所有动画纹理到第一帧
TextureAnimator.reset();
```

## 7. 全局函数

以下是与纹理系统相关的全局函数。

**saveTextures(lazy?)** — 保存所有纹理

```javascript
/**
 * 保存项目中所有未保存的纹理
 * @param {boolean} [lazy=false] - 如果为 true，没有本地文件的纹理不会被保存
 */
await saveTextures();
await saveTextures(true); // 仅保存有关联文件的纹理
```

**unselectTextures()** — 取消选中所有纹理

```javascript
/**
 * 取消选中项目中所有纹理的选中状态
 */
unselectTextures();
```

**loadTextureDraggable()** — 更新纹理列表拖拽功能（已废弃）

```javascript
/**
 * @deprecated 此函数不再存在
 */
// loadTextureDraggable();
```

```javascript
// 取消所有纹理的选中状态
unselectTextures();
```
