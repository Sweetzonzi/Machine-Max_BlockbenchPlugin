# math 函数

`math` 对象提供了数学计算功能，包括三角函数、指数函数、取整函数等。

## 三角函数

### math.sin(value)
计算角度的正弦值。

**参数**:
- `value` (double): 角度值（度）

**返回值**: 正弦值

**示例**:
```molang
math.sin(90)  // 返回 1.0
```

### math.cos(value)
计算角度的余弦值。

**参数**:
- `value` (double): 角度值（度）

**返回值**: 余弦值

**示例**:
```molang
math.cos(0)  // 返回 1.0
```

### math.tan(value)
计算角度的正切值。

**参数**:
- `value` (double): 角度值（度）

**返回值**: 正切值

### math.asin(value)
计算值的反正弦值（返回角度）。

**参数**:
- `value` (double): 值（-1.0 到 1.0）

**返回值**: 角度值（度）

### math.acos(value)
计算值的反余弦值（返回角度）。

**参数**:
- `value` (double): 值（-1.0 到 1.0）

**返回值**: 角度值（度）

### math.atan(value)
计算值的反正切值（返回角度）。

**参数**:
- `value` (double): 值

**返回值**: 角度值（度）

### math.atan2(y, x)
计算 y/x 的反正切值（返回角度）。

**参数**:
- `y` (double): Y坐标值
- `x` (double): X坐标值

**返回值**: 角度值（度）

**示例**:
```molang
// 在AE86悬挂动画中使用
math.atan2(subpart.connector_offset('connector.machine_max.right_front_wheel', 1)*16, 20)
```

## 指数和对数函数

### math.exp(value)
计算 e 的 value 次幂。

**参数**:
- `value` (double): 指数

**返回值**: e^value

### math.log(value)
计算值的自然对数（以 e 为底）。

**参数**:
- `value` (double): 值（> 0）

**返回值**: 自然对数

### math.pow(base, exponent)
计算 base 的 exponent 次幂。

**参数**:
- `base` (double): 底数
- `exponent` (double): 指数

**返回值**: base^exponent

### math.sqrt(value)
计算值的平方根。

**参数**:
- `value` (double): 值（≥ 0）

**返回值**: 平方根

**示例**:
```molang
// 在AE86悬挂动画中计算连杆长度
math.sqrt(math.pow(20-subpart.connector_offset('connector.machine_max.left_front_wheel', 1)*16, 2)+81)
```

## 取整函数

### math.floor(value)
向下取整。

**参数**:
- `value` (double): 值

**返回值**: 小于等于 value 的最大整数

**示例**:
```molang
// 在HUD速度显示中使用
math.floor((math.abs(vehicle.get('vehicle_speed')??0.0)) * 3.6 / 100)
```

### math.ceil(value)
向上取整。

**参数**:
- `value` (double): 值

**返回值**: 大于等于 value 的最小整数

### math.round(value)
四舍五入取整。

**参数**:
- `value` (double): 值

**返回值**: 最接近的整数

## 其他数学函数

### math.abs(value)
计算绝对值。

**参数**:
- `value` (double): 值

**返回值**: 绝对值

**示例**:
```molang
// 确保速度值为正数
math.abs(vehicle.get('vehicle_speed')??0.0)
```

### math.mod(value1, value2)
计算 value1 除以 value2 的余数。

**参数**:
- `value1` (double): 被除数
- `value2` (double): 除数

**返回值**: 余数

**示例**:
```molang
// 在HUD速度显示中提取十位数
math.floor(math.mod((math.abs(vehicle.get('vehicle_speed')??0.0)) * 3.6, 100) / 10)
```

### math.sign(value)
获取值的符号。

**参数**:
- `value` (double): 值

**返回值**: 1.0（正数）、-1.0（负数）、0.0（零）

### math.min(value1, value2)
返回两个值中的较小值。

**参数**:
- `value1` (double): 第一个值
- `value2` (double): 第二个值

**返回值**: 较小值

### math.max(value1, value2)
返回两个值中的较大值。

**参数**:
- `value1` (double): 第一个值
- `value2` (double): 第二个值

**返回值**: 较大值

### math.clamp(value, min, max)
将值限制在指定范围内。

**参数**:
- `value` (double): 要限制的值
- `min` (double): 最小值
- `max` (double): 最大值

**返回值**: 限制后的值

### math.lerp(start, end, alpha)
线性插值。

**参数**:
- `start` (double): 起始值
- `end` (double): 结束值
- `alpha` (double): 插值系数（0.0 到 1.0）

**返回值**: 插值结果

## 数学常量

### math.pi
圆周率 π 的值。

**类型**: 只读 double 值

**值**: 约等于 3.141592653589793

**示例**:
```molang
// 弧度转角度
radians * 180 / math.pi

// 角度转弧度
degrees * math.pi / 180
```

## 实际使用案例

### 1. 单位转换
```molang
// 米/秒转公里/小时
vehicle.get('vehicle_speed') * 3.6

// 弧度/秒转转/分钟（RPM）
vehicle.get('engine_speed') * 30 / math.pi
```

### 2. 动画控制
```molang
// 平滑动画插值
math.lerp(current_angle, target_angle, 0.1)

// 限制旋转角度在合理范围内
math.clamp(rotation_angle, -180, 180)
```

### 3. 物理计算
```molang
// 计算两点间距离
math.sqrt(math.pow(x2-x1, 2) + math.pow(y2-y1, 2) + math.pow(z2-z1, 2))

// 计算角度差
math.abs(math.mod(angle1 - angle2 + 180, 360) - 180)
```

## 注意事项

1. **角度单位**: 所有三角函数使用度为单位，而不是弧度
2. **性能考虑**: 复杂的数学计算可能影响性能，尽量简化表达式
3. **数值范围**: 注意函数的定义域和值域，避免数学错误
4. **精度问题**: 浮点数计算可能存在精度误差，重要计算应进行适当的舍入
