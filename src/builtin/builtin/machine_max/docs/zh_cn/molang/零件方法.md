# subpart 方法

`subpart` 对象提供了访问零件状态、连接器和子系统的方法。

## 信号相关方法

### subpart.get(channel)

获取零件信号存储中的值。

**语法**: `subpart.get(channel_name)`

**参数**:
- `channel_name` (字符串): 信号频道名称

**返回值**: 信号值，如果频道不存在则返回 `null`

**示例**:
```molang
// 获取零件特定信号
subpart.get('custom_signal')
```

## 连接器相关方法

### subpart.has_connector(connectorName)

检查零件是否拥有指定连接器，并且连接器是否连接了零件。

**语法**: `subpart.has_connector(connector_name)`

**参数**:
- `connector_name` (字符串): 连接器名称

**返回值**: 如果连接器存在且已连接零件则返回 1.0，否则返回 0.0

**示例**:
```molang
// 检查前轮连接器是否连接
subpart.has_connector('connector.machine_max.front_wheel')
```

### subpart.connector_offset(connectorName, axis)

获取连接器的偏移量（位置）。

**语法**: `subpart.connector_offset(connector_name, axis)`

**参数**:
- `connector_name` (字符串): 连接器名称
- `axis` (整数): 坐标轴 (0=X, 1=Y, 2=Z)

**返回值**: 指定轴的偏移量（米），如果连接器不存在则返回 0.0

**示例**:
```molang
// 获取前轮连接器的Y轴偏移（悬挂高度）
subpart.connector_offset('connector.machine_max.front_wheel', 1)
```

### subpart.connector_rotation(connectorName, axis)

获取连接器的旋转角度。

**语法**: `subpart.connector_rotation(connector_name, axis)`

**参数**:
- `connector_name` (字符串): 连接器名称
- `axis` (整数): 旋转轴 (0=X, 1=Y, 2=Z)

**返回值**: 指定轴的旋转角度（度），如果连接器不存在则返回 0.0

## 子系统相关方法

### subpart.has_subsystem(subsystemName)

检查零件是否拥有指定名称的子系统。

**语法**: `subpart.has_subsystem(subsystem_name)`

**参数**:
- `subsystem_name` (字符串): 子系统名称

**返回值**: 如果拥有该子系统则返回 1.0，否则返回 0.0

**示例**:
```molang
// 检查是否有发动机子系统
subpart.has_subsystem('subsystem.machine_max.engine')
```

### subpart.subsystem_durability(subsystemName)

获取指定子系统的当前耐久度。

**语法**: `subpart.subsystem_durability(subsystem_name)`

**参数**:
- `subsystem_name` (字符串): 子系统名称

**返回值**: 子系统耐久度，如果子系统不存在则返回 0.0

### subpart.subsystem_max_durability(subsystemName)

获取指定子系统的最大耐久度。

**语法**: `subpart.subsystem_max_durability(subsystem_name)`

**参数**:
- `subsystem_name` (字符串): 子系统名称

**返回值**: 子系统最大耐久度，如果子系统不存在则返回 0.0

### subpart.subsystem_active(subsystemName)

检查指定子系统是否处于激活状态。

**语法**: `subpart.subsystem_active(subsystem_name)`

**参数**:
- `subsystem_name` (字符串): 子系统名称

**返回值**: 如果子系统激活则返回 1.0，否则返回 0.0

### subpart.subsystem_destroyed(subsystemName)

检查指定子系统是否已被摧毁。

**语法**: `subpart.subsystem_destroyed(subsystem_name)`

**参数**:
- `subsystem_name` (字符串): 子系统名称

**返回值**: 如果子系统被摧毁则返回 1.0，否则返回 0.0

## 零件属性

### subpart.durability

零件的当前耐久度。

**类型**: 只读 double 值

### subpart.max_durability

零件的最大耐久度。

**类型**: 只读 double 值

### subpart.is_destroyed

零件是否已被摧毁。

**类型**: 只读 double 值（1.0=已摧毁，0.0=正常）

## 实际使用案例

### 1. 悬挂系统动画
在 AE86 动画中使用连接器偏移控制悬挂系统：

```json
{
  "axisRightFront": {
    "rotation": [0, 0, "math.atan2(subpart.connector_offset('connector.machine_max.right_front_wheel', 1)*16,20)"]
  },
  "rodLeftFront": {
    "rotation": ["-65.7725+math.atan2(20-subpart.connector_offset('connector.machine_max.left_front_wheel', 1)*16,9)", 0, 0]
  },
  "rodLeftFrontInner": {
    "scale": [1, "math.sqrt(math.pow(20-subpart.connector_offset('connector.machine_max.left_front_wheel', 1)*16,2)+81)/22", 1]
  }
}
```

**解释**:
- 使用 `subpart.connector_offset` 获取车轮连接器的Y轴偏移（悬挂高度）
- 通过三角函数计算悬挂连杆的角度和长度
- 实现真实的悬挂系统运动学

### 2. 子系统状态显示
在HUD中使用子系统状态控制警告灯：

```molang
// 发动机子系统状态检查
subpart.has_subsystem('engine') && subpart.subsystem_active('engine') ? 1.0 : 0.0

// 燃油子系统低油量警告
subpart.has_subsystem('fuel_tank') && subpart.subsystem_durability('fuel_tank') < 10.0 ? 1.0 : 0.0

// 发动机过热警告
subpart.has_subsystem('engine') && subpart.subsystem_destroyed('engine') ? 1.0 : 0.0
```

### 3. 条件动画控制
根据子系统状态控制零件动画：

```json
{
  "engine_cover": {
    "rotation": [
      "subpart.has_subsystem('engine') ? 45.0 : 0.0", 
      0, 
      0
    ]
  },
  "warning_light": {
    "scale": [
      "subpart.subsystem_destroyed('engine') ? 1.5 : 1.0", 
      "subpart.subsystem_destroyed('engine') ? 1.5 : 1.0", 
      "subpart.subsystem_destroyed('engine') ? 1.5 : 1.0"
    ]
  }
}
```

## 注意事项

1. **连接器命名**: 连接器名称通常遵循 `connector.machine_max.xxx` 的命名约定
2. **子系统命名**: 子系统名称通常遵循 `subsystem.machine_max.xxx` 的命名约定
3. **坐标系**: 连接器偏移和旋转使用局部坐标系
4. **性能优化**: 频繁调用的方法应考虑性能影响
