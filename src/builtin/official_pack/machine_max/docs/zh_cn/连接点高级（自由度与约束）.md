# 连接点高级（自由度与约束）

本教程目标：
- 在完成 fixed 入门后，学习 Advanced 连接点的自由度与约束配置。
- 理解 `joint_attrs`、弹簧阻尼、冲击参数的实际效果与调参方法。

---

## 1. 什么时候用 Advanced

适合这些场景：
- 车门铰链、可转向机构、摆臂、悬挂等需要“可动连接”的结构。
- 需要限制某些轴运动、放开某些轴运动的连接。

如果只是刚性拼装，优先继续使用 `Simple + fixed_*`。

---

## 2. 自由度与轴向

在关节里，常见约定：
- 轴 0~2：平移自由度（X/Y/Z）
- 轴 3~5：旋转自由度（X/Y/Z）

`joint_attrs` 的键通常按轴名组织（如 `x/y/z`，具体以 schema 为准），每轴可配置：
- `lower_limit`
- `upper_limit`
- `equilibrium`
- `stiffness`
- `damping`

---

## 3. 高级定义最小示例（铰链门思路）

下面示例展示“仅放开一个旋转轴”的思路（字段名以你的 schema 自动提示为准）：

```json
{
  "$schema": "../docs/zh_cn/schemas/part/subpart/connector/connector_static_attr.schema.json",
  "type": "Advanced",
  "direction": "xp",
  "integrity": 30.0,
  "joint_attrs": {
    "x": {
      "lower_limit": 0.0,
      "upper_limit": 0.0
    },
    "y": {
      "lower_limit": 0.0,
      "upper_limit": 0.0
    },
    "z": {
      "lower_limit": -70.0,
      "upper_limit": 5.0,
      "stiffness": 5.0,
      "damping": 1.0
    }
  }
}
```

调参逻辑：
1. 先只开限位，不加弹簧阻尼。
2. 动作确认正确后，再加小 `stiffness/damping`。
3. 最后再微调手感。

---

## 4. 软连接悬挂（思路模板）

目标：允许小范围位移并带回正趋势。

策略：
- 对平移轴给很小范围 `lower/upper`。
- 增加中低 `stiffness` + 中低 `damping`。
- 保持其余轴锁定，避免系统过度自由。

示例（概念模板）：

```json
"joint_attrs": {
  "y": {
    "lower_limit": -0.08,
    "upper_limit": 0.08,
    "equilibrium": 0.0,
    "stiffness": 3.0,
    "damping": 0.8
  }
}
```

---

## 5. 冲击参数与脱落机制

连接点断开不只看自由度，也受冲击参数影响：
- `integrity`：结构完整度上限。
- `impact_absorption`：冲击转为完整性损耗的比例。
- `impact_reduction`：冲击减免。
- `impact_multiplier`：冲击放大/缩小倍率。

调参建议：
1. 先保守值，确保不会轻易脱落。
2. 再逐步提高真实感（例如高载荷结构更易断）。

源码锚点：
- `AbstractConnector.handleAccumulatedImpact()`：冲击累计与断开逻辑。

---

## 6. 稳定性实践（重点）

推荐顺序：
1. 锁死全部轴，确认连接方向正确。
2. 只放开 1 个轴，验证运动方向。
3. 设置限位，再加低刚度低阻尼。
4. 小步增量调参，每次只改 1~2 个参数。

避免一次性把刚度/阻尼拉太高，否则容易出现：
- 抖动（振荡）
- 爆振（瞬间失稳）
- 锁死（运动异常受限）

`AbstractConnector.adjustJoint()` 内有稳定性保护和上限约束，说明“极端参数”会被限制，仍建议你从低值开始。

## 7. 高频问题排查

### 问题 A：门轴反向或开合方向怪异

处理：
- 先检查 `direction`。
- 再检查模型 locator 朝向。
- 最后再调整限位正负范围。

### 问题 B：一加弹簧阻尼就抖

处理：
- 同时降低 `stiffness` 与 `damping`。
- 缩小可动范围（`lower/upper`）。
- 先保证单轴可控，再扩展多轴。

### 问题 C：连接点太容易断

处理：
- 增加 `integrity`。
- 适度提高 `impact_reduction` 或降低 `impact_multiplier`。
