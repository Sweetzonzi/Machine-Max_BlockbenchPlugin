# Molang 文档

Machine-Max 使用 Molang 脚本系统来控制车辆动画、HUD显示和交互逻辑。Molang 是一种轻量级的脚本语言，专门用于实时计算和动画控制。

## 基本语法

Molang 表达式使用类似 JavaScript 的语法，支持基本的数学运算、条件判断和函数调用。

### 基本运算
```molang
// 算术运算
1.0 + 2.0 * 3.0  // 结果为 7.0
(1.0 + 2.0) * 3.0  // 结果为 9.0

// 比较运算
speed > 0  // 大于比较
durability <= max_durability  // 小于等于比较
v.a == 3.0  // 等于比较
```

### 三元表达式（条件运算符）
```molang
// 基本语法：条件 ? 真值 : 假值
variable > 0 ? 1.0 : 0.0

// 实际应用：根据子系统状态显示警告灯
subpart.has_subsystem('engine') && subpart.subsystem_destroyed('engine') ? 1.0 : 0.0

// 嵌套使用
v.fuel_level > 50 ? 0.0 : (v.fuel_level > 10 ? 0.5 : 1.0)
```

### 空值合并运算符（??）
```molang
// 基本语法：值 ?? 默认值
vehicle.get('signal_name')??0.0

// 实际应用：处理可能不存在的信号
math.abs(vehicle.get('vehicle_speed')??0.0) * 3.6

// 链式使用
vehicle.get('engine_speed')??vehicle.get('motor_speed')??0.0
```

### 函数调用
```molang
// 数学函数
math.sin(angle)
math.floor(value)

// 车辆和零件方法
vehicle.get('steering')
subpart.has_subsystem('engine')
```

## 常用模式

### 信号获取和安全访问
```molang
// 安全获取信号值
vehicle.get('signal_name')??default_value

// 条件检查后访问
subpart.has_subsystem('engine') ? subpart.subsystem_durability('engine') : 0.0
```

### 单位转换
```molang
// 米/秒转公里/小时
vehicle.get('vehicle_speed')??0.0 * 3.6

// 弧度/秒转转/分钟（RPM）
vehicle.get('engine_speed')??0.0 * 30 / math.pi
```
