# Molang 文档

Machine-Max 使用 Molang 脚本系统来控制车辆动画、HUD显示和交互逻辑。Molang 是一种轻量级的脚本语言，专门用于实时计算和动画控制。

## 目录结构

- [subpart 方法](./subpart.md) - 零件相关的方法和属性
- [vehicle 方法](./vehicle.md) - 车辆相关的方法和属性
- [math 函数](./math.md) - 数学计算函数
- [示例和案例](./examples.md) - 实际使用案例

## 基本语法

Molang 表达式使用类似 JavaScript 的语法，支持基本的数学运算、条件判断和函数调用：

```molang
// 基本运算
1.0 + 2.0 * 3.0

// 条件表达式
variable > 0 ? 1.0 : 0.0

// 函数调用
math.sin(angle)
subpart.has_subsystem('engine')
```

## 变量和上下文

Molang 表达式在不同的上下文中执行，可以访问不同的变量和方法：

- **动画上下文**: 在模型动画中使用，可以访问 `subpart` 和 `vehicle` 对象
- **HUD上下文**: 在HUD显示中使用，可以访问玩家实体、 `subpart` 和 `vehicle` 对象

## 常用模式

### 信号获取
```molang
// 获取车辆在某个频道的信号值
vehicle.get('signal_channel')

// 获取零件在某个频道的信号值
subpart.get('signal_channel')
```

### 条件动画
```molang
// 根据条件控制动画
subpart.has_subsystem('engine') ? 1.0 : 0.0
```

### 数学计算
```molang
// 单位转换（米/秒转公里/小时）
math.abs(vehicle.get('vehicle_speed')) * 3.6
```
